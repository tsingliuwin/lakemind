# 核心准则

跨行业普适的分析方法论。无论你在分析哪个行业、哪个主题，这些准则都必须遵守。

## 总则
* [总则：根本原则](general-principles.md) — 实事求是、数据为证、归属正确、最小假设、先理解后分析。

## 数据纪律红线
* [数据纪律](data-discipline.md) — 不可触碰的五条红线，违反即视为严重事故。

## 分则（按数据分析阶段）
* [数据画像](data-profiling.md) — 拿到数据后的第一步：了解时间范围、数据质量、字段含义。
* [数据清洗](data-cleaning.md) — 把原始数据变成高质量可直接分析的数据表。
* [数据分析](data-analysis.md) — 假设驱动、多维验证、统计陷阱规避。
* [数据呈现](data-presentation.md) — 图表选择、数字精度、结论表达规范。
