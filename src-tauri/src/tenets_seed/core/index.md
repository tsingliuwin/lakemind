# 核心准则

跨行业普适的分析方法论。无论你在分析哪个行业、哪个主题，这些准则都必须遵守。

## 总则
* [总则：根本原则](general-principles.md) — 实事求是、数据为证、归属正确、最小假设、先理解后分析、分析可复现。

## 数据纪律红线
* [数据纪律](data-discipline.md) — 不可触碰的五条红线，违反即视为严重事故。

## 数据安全准则
* [数据安全与隐私](data-security.md) — PII 识别、数据脱敏、数据分级、最小化原则、未成年人数据特化。纪律管"数字真实性"，安全管"数据使用合规"。

## 分则（按数据分析阶段）
* [数据画像](data-profiling.md) — 拿到数据后的第一步：时间范围、数据质量、字段含义、跨表一致性。
* [指标体系与口径](data-metrics.md) — 动手分析前先把指标定义清楚：五要素、口径文档化、同名不同义陷阱。
* [数据清洗](data-cleaning.md) — 把原始数据变成高质量可直接分析的数据表。
* [数据分析](data-analysis.md) — 假设驱动、多维验证、统计陷阱规避、时间序列陷阱。
* [数据呈现](data-presentation.md) — 图表选择、数字精度、结论表达规范。

## 准则变更准则
* [准则的准则](meta-governance.md) — 准则本身的修改也需要规范约束。
